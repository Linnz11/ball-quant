# 竞彩足球规矩

## 1. 单关 vs 串关 (最易错 — 今天就栽这)
- **串关(过关)=默认；单关=特权**，由体彩按 *场次 + 玩法* 逐一指定开放。
- **胜平负 / 让球胜平负**：单关**稀缺**——仅体彩选定的国际大赛场次开（世界杯正赛等）；日常联赛基本只能过关。
- **比分 / 总进球 / 半全场**：**常规即可单关**。
- **同场分裂资格**：同一场比赛，比分可单关、胜平负可能只能过关 —— 正常现象，非 bug。
- **铁律**：买前必查 `data-subactive` 的 **dg(单关)/gg(过关)** flag，per 场 per 玩法。`*dg:1`=单关可买；`*dg:0,*gg:1`=只能进串；`*dg:0,*gg:0`=没开售。
- **flag→玩法 映射（2026-06-16 live 验证）**：`nspf*`=胜平负、`spf*`=**让球**胜平负；`*dg`=单关、`*gg`=过关。**地面铁证** = 500 页面 `<span class="ico-dg">单关</span>` 角标**只挂在"不让球"(胜平负)行，让球行从来没有** → 让球默认串-only。**世界杯实测**：体彩给胜平负开了单关(~14 场, 163.com)，**让球一场都没开**(`spfdg:0` 全卡 12 场)。让球能否单买**全看体彩逐场指定，别假设"大赛=让球能单"**。
- 今日实证（WC）：比利时/乌拉圭 胜平负单关开；伊朗-新西兰 胜平负仅过关；西班牙 胜平负全关；**让球全部仅过关**；比分/总进球/半全场全单关开。

## 2. 串关机制
- 腿数：min **2**；max 按玩法 —— 胜平负/让球 **8串1**，总进球 **6串1**，比分 **4串1**，半全场 **4串1**。
- **混合过关**：只能**跨场**串不同玩法；**同一场比赛的不同玩法不能混进同一注串**（→ 一场要押多玩法 = 各自单关分票，或一场只贡献一个玩法给某串）。这正是"比赛内=一注"的**规则级强制**(§1.7)。
- 木桶原则：混合串腿数上限取所选玩法的**最低**上限。
- 复式 / 胆拖：支持。自由过关模式下不能同票兼做单关。

## 3. 五玩法分区 (统一 90 分钟基准)
- **胜平负**：胜 / 平 / 负。
- **让球胜平负**：让球**整数**、**主队视角**(主-N=主让球 / 主+N=主受让)、公布后不可改；让胜 / 让平 / 让负。
- **比分**：31 选项 = 主胜 13(1-0…5-2 + **胜其他**) / 平 5(0-0…3-3 + **平其他**) / 客胜 13(0-1…2-5 + **负其他**)。
- **总进球**：0,1,2,3,4,5,6,**7+** (双方进球和)。
- **半全场**：9 组合 (上半 45'含补 × 全场 90'含补，各 胜/平/负)。

## 4. 结算
- **全场 90 分钟(含伤停补时)，不含加时赛 / 点球。**（→ Poly 侧须独立核同口径，见 §1.7 结算等价。）
- **改期 / 弃赛**：单关**退票**；串关**剔除该腿、降档结算**（不退全本，奖金封顶随腿数降低）。
- **腰斩**：<36h 补赛按实际结果；>36h 无效（单关退、串关该腿按赔率1或剔除——源有分歧，以 sporttery.cn 为准）。
- **中立场**：照常结算，主客不变。

## 5. 限额
- **奖金封顶**：单关 **500万**；过关 2-3 场 **20万** / 4-5 场 **50万** / 6+ 场 **100万**（过关封顶反而**低于**单关）。
- **倍投** ≤ **99 倍** / 票 且 ≤ **¥20,000** / 票；基础 **¥2** / 注。
- **截止**：开赛前停售（实体店约开赛前 10 分钟）。
